

# Usage | Display Robot Description
## Display Model 
MJCF file, with Mojuco py:
```
python -m mujoco.viewer --mjcf path/to/elf3_prev_dof20/xml/scene.xml     
```
URDF file, with [yourdfpy](https://github.com/clemense/yourdfpy):
```
yourdfpy  path/to/elf3_prev_dof20/urdf/elf3_prev_dof20.urdf    
```
